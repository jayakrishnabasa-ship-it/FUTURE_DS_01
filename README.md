📊 Online Retail Sales Dashboard (Tableau)
📌 Project Overview
This project presents an interactive Sales Dashboard built using Tableau based on the Online Retail Dataset from Kaggle. The dashboard provides insights into sales performance, customer behavior, and product trends.

📂 Dataset
Source: Kaggle - Online Retail Dataset
The dataset contains transactional data of a UK-based online retail store.
🔑 Key Features:
InvoiceNo – Transaction ID
StockCode – Product ID
Description – Product Name
Quantity – Number of items purchased
InvoiceDate – Date of transaction
UnitPrice – Price per unit
CustomerID – Customer identifier
Country – Customer location

📈 Key Insights
A small number of products generate the majority of revenue.
Sales show seasonal trends with peak periods.
Most customers are concentrated in a few countries.
High-value customers contribute significantly to total sales.
Some transactions are cancelled, impacting overall revenue.
💡 Recommendations
Focus marketing efforts on high-performing regions.
Retain and reward loyal customers.
Promote best-selling products.
Improve inventory management to reduce cancellations.
Provide offers on low-performing products.
